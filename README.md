# Alpha Clash Pro

Alpha Clash Pro is a browser-based typing game that tests your speed and accuracy. Players match the currently displayed letter by pressing the correct key on their keyboard, earning points and losing lives for mistakes.

## Features

- Home screen with a play button
- Random letter prompt for fast typing practice
- Score tracking
- Life counter and game over screen
- Keyboard-themed UI built with Tailwind CSS and DaisyUI

## How to Run

1. Open `index.html` in a web browser.
2. Click **Play Now** or press **Enter** on the home screen.
3. Type the displayed letter as quickly as possible.
4. The game ends when lives reach zero.

## Controls

- `Enter`: Start the game from the home screen
- `Esc`: End the game immediately
- Letter keys: Match the currently displayed letter

## Project Structure

- `index.html` - Main game page
- `styles/style.css` - Custom font styling
- `scripts/alpha-clash.js` - Game logic and scoring
- `scripts/utiliti.js` - DOM utility functions
- `images/` - Background and icon assets
- `tailwind.config.js` - Tailwind configuration file (currently empty)

## Notes

- The game uses Tailwind CSS and DaisyUI from CDN links in `index.html`.
- For a polished experience, use a modern browser with keyboard support.
